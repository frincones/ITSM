## Servicios gestionados (MSP)

Mensual fijo. Incluye:
- Mesa de ayuda 8x5 (Standard) o 24/7 (Premium)
- Monitoreo proactivo de servidores y redes
- Backups gestionados
- Antivirus / EDR
- Patch management

Precios desde COP $1.500.000 / mes para empresas pequeñas (hasta 10 usuarios).
