# Preguntas frecuentes — Soporte TDX

## ¿Qué hago si mi computador no enciende?

Verifica primero: cable de poder bien conectado, batería con carga (si es laptop), tomacorriente con energía. Si nada funciona, lo más probable es fuente o batería. Reporta un ticket y te conectamos con un técnico.

## ¿Cómo creo un ticket de soporte?

Por esta llamada con Sofía. También puedes mandar mensaje a este mismo WhatsApp o email a soporte@tdxcore.com. Necesitas: tu nombre, empresa, descripción breve del problema, y urgencia (baja / media / alta / crítica).

## ¿Cuánto tarda la atención?

Depende del SLA de tu contrato:
- Standard: 8 horas hábiles
- Premium: 4 horas
- Enterprise: 1 hora

Para casos críticos (servidor caído, brecha de seguridad), siempre hacemos atención inmediata.

## No tengo internet en la oficina, ¿qué hago?

Primero: reinicia el módem y el router (apaga 30 segundos, prende). Si sigue, llama al ISP para confirmar si hay corte en tu zona. Si es problema interno (cableado, switch), reporta ticket urgente.

## ¿Hacen soporte remoto?

Sí. Usamos AnyDesk o TeamViewer. Te enviamos código de sesión y conectamos. Para empresas Premium/Enterprise el técnico puede ir on-site si el problema lo requiere.

## ¿Tienen servicio de respaldo (backup)?

Sí. Ofrecemos backup en cloud (AWS S3 / Azure Blob) y on-premise. Frecuencia: diaria, semanal, mensual según contrato. Restauración en horas según volumen.

## ¿Manejan ciberseguridad?

Sí: auditorías, hardening, EDR (Sophos, CrowdStrike), pentesting, capacitación a usuarios, plan de respuesta a incidentes.

## ¿Migran sistemas a la nube?

Sí. Hacemos assessment, plan de migración, ejecución y soporte post-migración. Cloud principal: AWS (somos partner), también Azure y GCP.
