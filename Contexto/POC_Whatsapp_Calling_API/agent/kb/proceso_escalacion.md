# Proceso de escalación TDX

## Cuándo escalar a humano

Sofía debe transferir al cliente con un asesor humano cuando:

1. **Cliente lo pide explícitamente** ("quiero hablar con una persona", "pásame con un asesor", "humano")
2. **Frustración detectada** (cliente repite, sube tono, dice "ya te dije", "no me entiendes")
3. **Tema fuera de su alcance** (queja formal, cancelación, devolución, tema legal)
4. **Caso crítico** (servidor caído de producción, brecha de seguridad activa, pérdida de datos)
5. **Más de 2 timeouts** sin respuesta del cliente

## A quién escalar según el tema

- **Soporte técnico**: equipo de mesa (lunes-viernes 8AM-6PM)
- **Ventas / nuevos clientes**: equipo comercial
- **Quejas / cancelaciones**: gerencia de cuenta
- **Pagos / facturación**: equipo administrativo
- **Emergencia 24/7**: ingeniero de guardia

## Información que debe pasar Sofía al transferir

1. Nombre del cliente (si lo tiene)
2. Empresa
3. Resumen del problema
4. Lo que ya se intentó (si hubo intentos previos)
5. Urgencia percibida (baja / media / alta / crítica)
